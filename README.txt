Sistema que lê dados do publisher, "UFSC/DAS/NEMO", e os apresenta numa taixa de 5 segundos. Os dados são representados em texto, à esquerda.
Também apresenta dados das APIs: https://philosophyapi.pythonanywhere.com/api/philosophers/
https://pokeapi.co/
https://web.dragonball-api.com/
A primeira é usada para apresentar uma frase de um filósofo aleatório e seu nome cada vez que o sistema é inicado.
As seguintes expõem uma imagem e nome de um personagem e nome da série correspondente, de maneira semelhante à frase.
Foi usado CustomTkinter para a Interface Gráfica, PIL (PILLOW) para gerenciar imagens e requests para acessar urls.
